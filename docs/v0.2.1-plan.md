# PyFixAgent v0.2.1 Plan

v0.2.1 is a small CLI and project hygiene release. It does not add new Agent intelligence.

## Scope

- Add argparse options for config, workspace, task, initial mode, context strategy, and max iterations.
- Make configuration precedence explicit: CLI arguments > config file > code defaults.
- Keep default behavior aligned with `configs/default.yaml`: replacement mode and traceback context.
- Reset example workspaces from the main repository baseline instead of nested Git repositories.
- Keep generated patches, traces, caches, temp files, and secrets out of Git.

## Example Commands

```bash
python -m pyfixagent.main --help
python scripts/reset_demo.py --all
python -m pyfixagent.main --workspace workspaces/demo_project --mode replacement --context-strategy traceback
python -m pyfixagent.main --workspace workspaces/sklearn_iris_tree_project --max-iterations 3
```

## Out Of Scope

No Web UI, Docker sandbox, GitHub integration, RAG, multi-model voting, generated tests, AST editor, complex import graph, or large benchmark system.
